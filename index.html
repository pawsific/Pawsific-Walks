Pawsific Daily Walks
Connect your Google Calendar to see every walk — Rover and direct — in one place.

One-time setup

1
Go to console.cloud.google.com, create a project, enable the Google Calendar API, then create an OAuth 2.0 Client ID for a Web application.
2
In the OAuth client settings, add file:// (or the URL you open this file from) to Authorized JavaScript origins, and the same URL to Authorized redirect URIs.
3
Paste your Client ID below — it looks like 123456….apps.googleusercontent.com. This is saved in your browser only.
Your calendar data never leaves your browser. The Client ID is stored locally on this device only.
