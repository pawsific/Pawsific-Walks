Pawsific Daily Walks
Connect your Google Calendar to see every walk in one place.

One-time setup

1
Go to console.cloud.google.com, create a project, enable the Google Calendar API, then create an OAuth 2.0 Client ID for a Web application.
2
Add https://pawsific.github.io to Authorized JavaScript origins.
3
Paste your Client ID below — saved in your browser only.
Your data never leaves your browser. Client ID is stored locally only.
